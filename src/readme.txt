Readme
-------
What's about?
This script extracts scintillations info (S4 & sigma_phi) from an ISMR file. Then, data is
arranged in the LISN or modified LISN format.  
-------
Instructions:

1) Place the ismr data into "~/data_input/Data_set/" folder
2) Run ismr2S4.py (modified LISN format)
2) Run ismr2S4-LISN.py (LISN format)
3) The output data: scintillation file(*.s4), will be stored in "~/data_output" folder
